# A simple Boggle solver

Run it to generate and solve a board, or supply the letters of a board to solve.

    $ ./boggle.py
    $ ./boggle.py <letters>

For example:

    $ ./boggle.py oviiahadnilesubo

    o  v  i  i
    a  h  a  d
    n  i  l  e
    s  u  b  o
